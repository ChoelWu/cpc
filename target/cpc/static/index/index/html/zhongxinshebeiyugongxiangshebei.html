中心设备与共享设备

2005年学院独立设置了中学化学教学实验室，2008年划归教育学院，由教育学院负责实验室的教学和管理工作，2017年起重新回到化学化工学院，学院负责实验室的管理和教学工作。在教育学院管理的近十年时间里，实验室的设备没有得到及时更新，部分设备老化；同时，实验课程体系也没有及时适应社会发展需要，相对陈旧。学院接管后，重新成立了中学化学教学实验中心，组建了中心机构，密切联系中学教育教学实际，与教育机构和中学加强了联系和沟通，更新了部分设备，修订了实验教学课程体系，目前中心运行良好，教学质量逐年提高，成为学院学生最喜欢的实验教学体系之一。中心学生实验的环境更宽敞，使用效率更高，大大改善了化学实验教学中心的实验条件和环境。突出我校“师范教育”的特色，实验室环境与安全建设做到科学、规范、整齐、简洁、高效。中心共有实验室7间，面积500平方米。各类仪器设备400件，价值100万元。实验设施和环境完全达到了教育部有关规定。

西北师范大学化学化工学院分析测试中心成立于1985年9月，依托化学化工学院面向全校，辐射甘肃，以服务教学、科研为基本建设目的。1987年中心评为国家教委系统先进实验室集体，而后得到省教委、省科委、学校的多次表彰。中心人员承担或参与完成了多项国家级和省级科研项目。截止目前，测试中心现有各类仪器设备3735台（件），总价值7959.94万元。现拥有超导核磁共振波谱实验室、高分辨液-质联用实验室、圆二色光谱实验室、激光拉曼光谱实验室、高温GPC/多角度激光散射实验室、热分析实验室、比表面/X-射线粉末衍射实验室、原子力扫描显微镜实验室、元素分析实验室、原子吸收/原子荧光光谱实验室、付立叶红外光谱实验室、紫外/荧光光谱实验室等二十余个专业实验室，配备多台大型精密分析测试仪器，建筑面积达到4000多平方米，是目前西北地区近代分析测试手段较为齐全的化学检测分析中心之一。分析测试中心实现了开放共享，所有学生均可通过预约系统进行测样、参观、认识等学习活动。
