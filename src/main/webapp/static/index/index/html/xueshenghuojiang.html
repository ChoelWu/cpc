学生获奖

在实践能力和创新能力的实践过程中，学院化学(师范)专业学生先后参加学校、甘肃省、国家等不同层次的学科竞赛和教学技能大赛，仅2017年度共有19人次学生荣获国家和省部级竞赛奖励。如获第五届全国高等师范院校大学生化学实验邀请赛二等奖2项、三等奖1项，西北地区高等院校师范生教学技能大赛化学组二等奖，甘肃省大学生化学竞赛团体二等奖，甘肃省三好学生3人，西北师范大学第四届创业计划大赛二等奖1项、三等奖2项等，取得了较好的成绩，彰显了学院师范生较强的实践和创新能力，得到了学校、社会的认可。
