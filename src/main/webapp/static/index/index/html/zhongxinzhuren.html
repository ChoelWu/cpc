中心主任

个人简介：权正军，男，博士，教授，甘肃省飞天学者特聘教授。参研的项目获甘肃省自然科学奖二等奖、三等奖各一项（第二作者）和甘肃省高校科技进步一、二等奖各2项。获甘肃省五四青年奖章提名奖、西北师范大学首届“教学科研”之星和教学名师等荣誉称号。

主要从事有机药物小分子的合成和杂环化合物的合成工作。研究方向主要有光电催化杂环有机化合物合成和含磷有机化合物的合成。近五年以第一作者/通讯作者在Chem. Chem.，Adv. Synth. Catal.，Chem Rec，Catalysis Science & Technology等期刊发表论文40余篇；授权专利4件。

教学相关工作：近年来主讲本科生《有机化学》、《物理化学及实验》、《中学化学实验教学研究》，每年承担研究生《有机合成化学》等课程。主持校级研究项目“非化学专业化学系列课程改革的研究与实践”和“研究生有机合成化学教学实践与改革探索”两项。

现为中国化学会会员，甘肃省青年联合会第十届委员会常务委员，兼任甘肃省化学会副理事长和甘肃省教育学会化学教学专业委员会副理事长等职务。应邀在第十七届全国高等师范院校化学课程结构与教学改革研讨会、甘肃省大学生化学竞赛、兰州城市学院化学化工学院举办首届中学校长论坛以及其他兄弟院校的实验教学示范中心的座谈会上作大会发言及分会报告，介绍教学改革经验和体会，得到了与会代表的肯定和好评。
