中心成员简介

中心有一支结构合理、学科齐全、教师教育能力突出的师资队伍，现有教师19人，其中国家“万人计划”领军人才1人，正高级7人，副高级12人，5名中学老师等基础教育工作者，既有化学学科专业教师，还有教育学、心理学和化学教学论等教师教育专业教师，密切高校和中学基础教育部门的合作办学，聘请中学教师为学生授课，并担任教育硕士的校外指导教师和教育实习校外指导教师。实验中心主任聘任各实验课的主讲教师和任课教师。主讲教师全面负责实验课的教学与管理。实验指导教师在全院教师中公开招聘，择优聘任；各项工作责任到人。实验技术人员全部实行公开招聘、竞争上岗。
